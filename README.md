# DDoS Awareness Demo Website

## 📌 Project Overview
This project is an educational website created to explain Distributed Denial of Service (DDoS) attacks and their prevention methods.

The website is designed for academic demonstration purposes only.

## 🎯 Objective
The goal of this project is to:
- Explain what a DDoS attack is
- Show how DDoS attacks work (conceptually)
- Provide prevention and mitigation strategies
- Support cybersecurity awareness learning

## 🚀 Features
- Clean cyber-themed homepage
- Explanation of DDoS concepts
- Separate page for how DDoS works
- Prevention strategies section
- Responsive design

## 🛠 Technologies Used
- HTML5
- CSS3
- GitHub Pages (for hosting)
